# Industrial Python Automation
This repository contains production-ready Python scripts for data acquisition and protocol bridging in Industrial IoT (IIoT) environments.

### Key Components:
- **Modbus Monitor:** Scripts to poll data from RS485/Modbus RTU sensors.
- **Protocol Bridge:** High-performance service to transform raw hardware data into JSON payloads for MQTT brokers.
- **Error Handling:** Implemented robust logging and exception handling for industrial reliability.

### Use Case:
Bridge between legacy industrial hardware and modern cloud/dashboard systems.
